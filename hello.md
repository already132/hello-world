## git 명령어 요약

- clone : 원격 저장소 복사 
- add : 스테이지 영역에  작업 파일 추가
- commit : 스테이지 영역의 파일을 가지고 커밋 (*세이브를) 만든다
- push : 원격 저장소에 커밋을 업로드 

-  사실 아직 내용을 
-  무엇 올려야 할지

## checkout
- 브랜치를 선탁하고 작업을 수행 할 때 체크아수을 해야 한다. 소스트리에서는 브렌치를 클릭하는 것으로 체크 아웃을 수행 할 수 있다.
